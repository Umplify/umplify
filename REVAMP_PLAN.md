# Umplify website revamp brief

## Objective
Reposition Umplify from a cloud-first consultancy to an AI transformation and agentic systems partner, while preserving the company's engineering credibility in Azure, application modernization, DevOps, and cloud architecture.

## Proposed positioning
Umplify helps businesses become AI-ready through agentic workflows, enterprise integrations, and the cloud/platform engineering required to operate securely in production.

## Proposed message hierarchy
1. AI Transformation
2. Agentic Workflows and MCP-oriented integrations
3. Cloud Platform Engineering
4. Application modernization and delivery excellence

## Proposed navigation
- Home
- AI Transformation
- Cloud Platform Engineering
- About
- Contact

- `/ai-transformation/`
- `/home-v2/`
- `/cloud-platform-engineering/`

## Why these are drafts
The current connector path allowed safe branch creation and new-file creation, but did not expose a clean direct overwrite path for the existing tracked pages in this repo. These draft pages are intended for review and can then be folded into the existing routes in a follow-up commit or manual merge workflow.

## Recommended next production changes
- Replace the current homepage copy with the AI-first homepage draft
- Add AI Transformation to the main navigation
- Reframe Cloud Solutions as Cloud Platform Engineering
- Rewrite About around AI-readiness plus senior engineering execution
- Refresh Contact around discovery calls and assessment-led conversion
- Update global site metadata in `_config.yml` to reflect the new positioning
